Core Features

Advanced Waypoint Management

Create and manage waypoints with multiple coordinate formats, NOTAM integration, and file attachments. Import/export support for CSV, KML, and ForeFlight.

Interactive Deconfliction

Track aircraft operations with embedded maps and filtering. Automatic hourly exports to ForeFlight with email notifications for stakeholders and overlapping operations.

Contractor Management

Grant temporary access to contractors with controlled permissions. Monitor activity and manage program-specific access.

Advanced Data Management

Organize waypoints, plates, and map layers into distinct programs. Mass import NOTAMs and waypoints with dedicated CSV formats and automatic formatting.

Mobile Optimized

Install as a Progressive Web App for offline access. Touch-optimized interface with persistent settings.

Seamless Integration

Direct ForeFlight integration with content packs. Optional Dropbox sync for automatic file management.

Email Notifications

Manage email lists for stakeholder notifications. Send deconfliction updates to external parties without requiring system access.

Advanced Filtering

Powerful filtering and search capabilities for deconfliction entries. Sort by date, aircraft type, mission, and operational status.
